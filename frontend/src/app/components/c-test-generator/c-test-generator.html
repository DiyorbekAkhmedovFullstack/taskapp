C-Test Generator

Wie funktioniert ein C-Test?

Geben Sie einen deutschen Text mit mindestens 2 Sätzen ein. Der erste Satz bleibt unverändert, ab dem zweiten Satz wird bei jedem zweiten Wort die zweite Hälfte entfernt.

{{ errorMessage }}

Beispieltext:

"Deutschland ist ein Land in Europa. Es h__ vi___ int_____ St____ u__ ei__ ri____ Ku____. Vi___ Me_____ le____ do__ u__ st_____ De____."

So würde ein generierter C-Test aussehen.
